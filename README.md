## 製品概要
小学生以下の子供に向けた、実践型のSNS利用法勉強アプリ
### 背景(製品開発のきっかけ、課題等）
SNSが生活から切り離せない存在になった今、SNSの危険性をしらないままに利用してしまう子供が増えている。
そこで、間違ったSNSの使い方をして子供が危険な目に合ってしまう前に、実際のSNSに近い環境で練習できるアプリを作りたいと考えた。
対AIであるため、失敗が許容されており、また相手(AI)からの反応を通じて、適切なネット上での投稿、コミュニケーションの取り方を能動的に身につける機会が得られる。

### 製品説明（具体的な製品の説明）
新規登録・ログインをすることで、利用が可能。
実際のSNSのようなタイムラインが存在し、投稿、いいね、コメントができる。
AIが自分の投稿にコメントし反応してくれる(Geminiを利用)。
投稿の内容によってはバスりや炎上が疑似的に発生する。
ネット上での拡散の速さや、反応が必ずしも肯定的とは限らないことを実感をもって学べる。


### 特長
#### 1. 特長1　実際のSNSのように自由に投稿可能！
#### 2. 特長2　AIから、子供向けの適切なコメント・リアクションをもらえる！
#### 3. 特長3　

### 解決出来ること
SNSを安心・安全に使いたい子供やそれを支援したい親
SNSの使い方を学び直したい人に学びなおしのきっかけを提供
### 今後の展望
AIによる自動投稿を実装し、利用者がその投稿にコメントできるようにする。
より実際のSNSに近づけた機能を追加していく（画像や動画の投稿機能、DMなど）。
投稿前に自分の文章を検閲してくれるモードを追加し、自由に書き込みできる（検閲なし）モードと切り替えられるようにする。

### 注力したこと（こだわり等）
* Geminiを利用することで、利用者の投稿内容を適切な内容か不適切な内容か判断し、リアクションを変える。
  具体的には、適切(健全)な内容であったらバズり、不適切な内容であったら炎上...？（現実のSNSとは異なり、ここではSNSの使い方の学習が目的のため、投稿について優しい言葉で注意を受ける）
  することでSNSで起こりうるトラブルも体験できるようにした点。
